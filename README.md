## 项目目录

- [project1](/project1) - OpenCV基础操作实验
- [project2](/project2) - 图像几何变换与频域分析
  
## 环境配置

推荐使用Python 3.11+与OpenCV 4.10+版本组合（最新稳定版为4.11.0）：

```bash
pip install opencv-python==4.11.0
pip install opencv-contrib-python==4.11.0  # 包含扩展模块
```

## 学习路径
